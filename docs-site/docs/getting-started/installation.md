---
title: Instalación
---

1. Clona el repositorio.
2. Instala dependencias desde la raíz del proyecto:

   ```bash
   npm install
   ```

3. Verifica que exista la carpeta `data/`. La base de datos se crea en `data/museo.sqlite` cuando inicias la app.

Variables de entorno:

- `PORT`: puerto del servidor. Por defecto es `3000`.
