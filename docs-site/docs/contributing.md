Gracias por contribuir al proyecto.

## Desarrollo local

```bash
npm install
npm start
```

## Documentación (Docusaurus)

Desde `docs-site`:

```bash
npm install
npm run start
npm run build
npm run serve
npm run deploy
```

Si vas a abrir un PR, por favor actualiza también la documentación cuando sea necesario.
