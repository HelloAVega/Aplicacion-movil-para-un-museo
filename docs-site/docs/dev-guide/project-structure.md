---
title: Estructura del proyecto
---

- `index.html`: interfaz principal.
- `styles.css`: estilos responsive y modo oscuro.
- `app.js`: lógica del frontend.
- `server.js`: API REST y persistencia.
- `data/`: base de datos SQLite (`museo.sqlite`).
- `Dockerfile` y `docker-compose.yml`: despliegue con contenedores.
- `docs-site/`: documentación en Docusaurus.
